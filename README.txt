Katalyst WordPress Theme
========================

Theme type: hybrid classic theme with theme.json support for WordPress 6.9.x.

Included:
- front-page.php landing page based on KATALYST Landing.html
- reusable template parts for all homepage sections
- blog, archive, search, 404, page, and single templates
- two custom page templates based on the wireframe alternatives
- theme.json palette, spacing, and typography settings
- JS for feed filtering, reveal animations, and mobile navigation

Setup:
1. Copy this directory into wp-content/themes/katalyst.
2. Activate the theme in WordPress.
3. Optionally assign a static front page and a posts page.
4. Create a menu and assign it to the Primary Navigation location.
5. Publish posts to replace the fallback feed/news content automatically.

Assets:
- Existing logos in /assets are reused directly.
- Original design source files remain in the theme root for reference.
